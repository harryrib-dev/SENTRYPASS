# 🛡️ SentryPass

O **SentryPass** é um gerador de senhas moderno e ultrasseguro, desenvolvido para oferecer máxima entropia e proteção criptográfica diretamente no navegador. 

A aplicação foi construída com foco em **segurança**, **responsividade** e **experiência do usuário (UX)**.

## 🚀 Funcionalidades

* **Segurança de Nível Bancário:** Utiliza a `window.crypto.getRandomValues()` para garantir aleatoriedade real, superior ao `Math.random()`.
* **Controle Total:** Ajuste o comprimento (mínimo de 8 caracteres) e escolha entre letras (abc/ABC), números e símbolos.
* **Histórico Inteligente:** Mantém as últimas 3 senhas geradas visíveis para facilitar o fluxo de trabalho.
* **Mobile-First:** Design totalmente responsivo que funciona perfeitamente em smartphones, tablets e desktops.
* **Cópia Rápida:** Botão dedicado para copiar a senha para a área de transferência com um único toque.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estrutura semântica.
* **CSS3:** Design moderno com variáveis CSS, Grid e Flexbox.
* **JavaScript (ES6+):** Lógica de geração e manipulação de DOM.
* **Web Crypto API:** Para geração de números aleatórios criptograficamente seguros.

## 📱 Como usar

1.  Acesse o projeto pelo [[Link]](https://harryrib-dev.github.io/SENTRYPASS/).
2.  Defina o tamanho da senha usando os botões `+` e `-`.
3.  Marque os tipos de caracteres desejados.
4.  Clique em **GERAR NOVA**.
5.  Clique em **COPIAR** para usar sua senha em qualquer lugar.
